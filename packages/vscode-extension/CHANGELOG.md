# Change Log

All notable changes to the "PROJAX for Editors" extension will be documented in this file.

## [3.0.0] - 2024-01-XX

### Added
- Initial release of PROJAX for Editors
- Project list view with search and sorting
- Project details view with full feature parity to desktop app
- Auto-detection of current workspace project
- Dual connection mode (API + direct database fallback)
- Script running and process management
- Tag management
- Test file scanning
- Port and URL detection
- Flexible project opening options
