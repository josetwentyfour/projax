# Prxi Keyboard Shortcuts

Complete reference for all keyboard shortcuts in Prxi.

## Navigation

### Project List

- `↑` / `k` - Move up in project list
- `↓` / `j` - Move down in project list
- `Enter` - Select project (updates details panel)

### Panel Navigation

- `Tab` - Switch between project list and details panel
- `←` / `Left Arrow` - Move focus to project list
- `→` / `Right Arrow` - Move focus to details panel

### Scrolling

- `↑` / `k` - Scroll up
- `↓` / `j` - Scroll down
- `Page Up` - Fast scroll up (if supported)
- `Page Down` - Fast scroll down (if supported)
- `Home` - Jump to top (if supported)
- `End` - Jump to bottom (if supported)

## Actions

### Project Actions

- `s` - Scan selected project for tests
- `p` - Scan ports for selected project
- `r` - Show available scripts
- `x` - Stop all scripts for selected project

### General Actions

- `?` - Show help screen
- `q` / `Esc` - Quit Prxi

## Help Screen

When help is shown (`?`):
- `?` - Hide help
- `q` / `Esc` - Quit Prxi

## Tips

### Vim Users

If you're familiar with vim:
- `j` = down (like vim)
- `k` = up (like vim)
- Navigation feels natural

### Arrow Key Users

If you prefer arrow keys:
- All navigation works with arrow keys
- No need to learn vim bindings
- Arrow keys work everywhere

### Quick Actions

Common workflows:
1. `j`/`k` to navigate
2. `s` to scan
3. `r` to see scripts
4. `q` to quit
